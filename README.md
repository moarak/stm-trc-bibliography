
## Structural topic model analysis of Peru's Truth and Reconciliation Commission bibliographic metadata (2003-2024)

[![DOI](https://zenodo.org/badge/DOI/10.5281/zenodo.19955124.svg)](https://doi.org/10.5281/zenodo.19955124)

This repository is a research compendium containing the R code, 
session information, and outputs for a structural topic model 
(STM) analysis of bibliographic metadata related to Peru's Truth 
and Reconciliation Commission (TRC) and its Final Report. 
It is intended to make the analysis fully reproducible. 

The underlying dataset is archived on Zenodo: 
[Bibliographic metadata on Peru's Truth and Reconciliation Commission: 
A curated dataset (2003–2024)](https://doi.org/10.5281/zenodo.19420345)

## Repository structure

```
.
├── R/                        # Analysis scripts (run in order)
│   ├── 01_load_data.R        # Load and verify raw dataset
│   ├── 02_eda_metadata.R     # Exploratory analysis of bibliographic metadata
│   ├── 03_prepare_stm_input.R  # Construct text variable and covariates
│   ├── 04_process_text_stm.R   # Text normalisation and STM corpus preparation
│   ├── 05_fit_stm_model.R    # K selection, model fitting, and diagnostics
│   └── 06_interpret_topics.R # Topic interpretation and figure generation
├── data/
│   ├── raw/                  # Original dataset (download from Zenodo — see below)
│   └── processed/            # Intermediate files (generated by pipeline)
├── output/
│   ├── figures/              # Figures generated by the pipeline
│   └── tables/               # Tables generated by the pipeline
├── docs/                     # Codebook and supplementary documentation
├── DESCRIPTION               # Project metadata and dependencies
└── README.md
```

## Data

The raw dataset is not included in this repository. Download 
`peru_trc_bibliography_2003_2024.csv` from the Zenodo deposit 
(link above) and place it in `data/raw/` before
running the scripts.

Processed data and model objects are not included either — they are
regenerated by running the pipeline from scratch.

## How to reproduce

1. Clone this repository
2. Open `stm-trc-bibliography.Rproj` in RStudio
3. Place the raw dataset in `data/raw/` (see Data section above)
4. Run `renv::restore()` in the RStudio console to install the required packages
5. Run the scripts in `R/` in numbered order, or run all at once:

```r
source("R/01_load_data.R")
source("R/02_eda_metadata.R")
source("R/03_prepare_stm_input.R")
source("R/04_process_text_stm.R")
source("R/05_fit_stm_model.R")
source("R/06_interpret_topics.R")
```

## Software environment

R version and package versions are recorded in `renv.lock`. Run
`renv::restore()` to reproduce the exact environment used in the analysis.
